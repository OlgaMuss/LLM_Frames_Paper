Learning material used:  
A microcontroller is a small computer on a chip that can execute a simple program.  
A microcontroller must be connected to a positive and a negative terminal.  
To control other devices, a microcontroller can output a voltage of 3 V on its pins.  
It can switch this voltage on and off for different pins, also for multiple pins at the same time.  
When a pin is switched on, it is "HIGH"; when it is switched off, it is "LOW".  
This switching between "HIGH" and "LOW" does not happen randomly — it is controlled by a program, such as Blockly for Marty.  
Microcontrollers use a programming language called C++.
