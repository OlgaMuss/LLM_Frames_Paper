Ein Mikrocontroller ist ein kleiner Computer auf einem Chip, der ein einfaches Programm ausführen kann.
Ein Mikrocontroller muss an einen Plus- und einen Minuspol angeschlossen werden.
Um andere Geräte zu steuern, kann ein Mikrocontroller an seinen Pins eine Spannung von 3 V ausgeben.
Er kann diese Spannung für verschiedene Pins an- und ausschalten, auch für mehrere Pins gleichzeitig.
Wenn ein Pin eingeschaltet ist, ist er „HIGH“; wenn er ausgeschaltet ist, ist er „LOW“.
Dieses Umschalten zwischen „HIGH“ und „LOW“ geschieht nicht zufällig – es wird von einem Programm gesteuert, wie beispielsweise Blockly für Marty.
Mikrocontroller verwenden eine Programmiersprache namens C++.
