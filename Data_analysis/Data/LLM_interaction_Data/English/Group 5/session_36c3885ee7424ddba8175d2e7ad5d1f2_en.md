- **Learning Material:**  A microcontroller is a small computer on a chip that can run a simple program.
A microcontroller must be connected to a positive and a negative pole.
To control other devices, a microcontroller can output a voltage of 3 V on its pins.
It can switch this voltage on and off for different pins, also for several pins at the same time.
When a pin is switched on, it is "HIGH"; when it is switched off, it is "LOW".
This switching between "HIGH" and "LOW" does not happen randomly — it is controlled by a program, for example Blockly for Marty.
Microcontrollers use a programming language called C++.
